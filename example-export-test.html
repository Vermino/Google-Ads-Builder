What to Check After Export

  • File Name: Should be google-ads-campaign-YYYY-MM-DD.csv
  • Column Count: Should have 31 columns
  • Match Types: Exact [brackets], Phrase "quotes", Broad (no formatting)
  • CPC Calculations: Verify match type modifiers are applied correctly
  • Headlines: All 15 headline columns present (some may be empty)
  • Descriptions: All 4 description columns present
  • Character Encoding: UTF-8 with BOM (special chars display correctly)
  • CSV Escaping: Commas and quotes in text should be properly escaped
