## 8. Responsive Behavior

### Desktop (> 1024px)
- Cards in 3-column grid
- Full toolbar with all buttons visible
- Checkboxes 16x16px
- Modal width: max-w-lg (512px)

### Tablet (768px - 1024px)
- Cards in 2-column grid
- Toolbar buttons slightly smaller
- Checkboxes 16x16px
- Modal width: max-w-md (448px)

### Mobile (< 768px)
- Cards in 1-column stack
- Toolbar buttons stack vertically or overflow menu
- Checkboxes 20x20px (larger for touch)
- Modal width: 90% of screen
- Padding adjusted for smaller screens

## Visual Design Principles

1. **Clarity**: Clear visual feedback for all states
2. **Consistency**: Same patterns across ad groups and ads
3. **Feedback**: Immediate response to user actions
4. **Safety**: Destructive actions clearly marked in red
5. **Efficiency**: Minimal clicks to complete tasks
6. **Accessibility**: High contrast, keyboard support
